
## Learning Management System (LMS)

**Role:** Software Engineer
**Organization:** DigiTechnology Nepal

### Problem

The existing system struggled with performance and scalability during online exams, particularly under high concurrent user load.

### Solution

* Designed and developed a scalable **Learning Management System (LMS)** with optimized database architecture.
* Implemented efficient backend APIs to handle authentication, exam workflows, and user management.
* Separated long-running processes to prevent server overload during peak usage.

### Technologies Used

* Backend: Node.js, Python
* Frontend: React.js
* Database: MongoDB and SQL-based relational database
* Infrastructure: Optimized server-side architecture

### Impact

* Supported **1,000+ concurrent users** during online examinations.
* Improved system stability and responsiveness under heavy load.
* Reduced downtime and improved user experience for students and administrators.
