## AWS Certified Developer – Associate

**Issuer:** Amazon Web Services (AWS)
**Validity:** March 2025 – March 2028

* Validates hands-on experience developing and maintaining applications on AWS.
* Demonstrated proficiency in:

  * AWS SDK usage
  * IAM roles and permissions
  * Serverless services (Lambda, API Gateway)
  * Debugging and optimizing cloud-based applications
* Applied certification knowledge in real-world projects involving scalable APIs and cloud deployments.

## AWS Certified Machine Learning Engineer – Associate

**Issuer:** Amazon Web Services (AWS)
**Validity:** August 2025 – August 2028

* Validates ability to build, deploy, and monitor machine learning solutions on AWS.
* Demonstrated proficiency in:

  * Data preparation and feature engineering
  * Model training and evaluation
  * ML deployment and monitoring
  * Integrating ML models into production systems
* Relevant to projects involving **Generative AI**, **RAG pipelines**, and applied machine learning.
